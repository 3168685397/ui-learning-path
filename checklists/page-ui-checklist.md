# 页面 UI 检查表

每次做完一张页面，用这个检查。

## 5 秒测试

- [ ] 5 秒内能看出页面主题。
- [ ] 5 秒内能读清主标题。
- [ ] 5 秒内知道能点击哪里。

## 黑白风格

- [ ] 页面没有不必要的彩色。
- [ ] 至少有 4 层灰度。
- [ ] 背景没有压过文字。
- [ ] 按钮和文字不会糊在一起。

## 高级感

- [ ] 页面没有塞满。
- [ ] 标题、按钮、说明有明确对齐关系。
- [ ] 字体不超过 2 种。
- [ ] 装饰不抢主内容。

## 宇宙背景

- [ ] 有空间感或尺度感。
- [ ] 中间有放标题的位置。
- [ ] 视频运动不太快。
- [ ] 转黑白后仍然好看。

## 手机端

- [ ] 标题没有溢出。
- [ ] 按钮可以点击。
- [ ] 文字没有被背景吃掉。
- [ ] 首屏不需要横向滚动。
