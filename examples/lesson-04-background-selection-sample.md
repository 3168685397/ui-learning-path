# Lesson 04 背景选择示范

## 作业信息

```text
课程：Lesson 04
自评分：82/100
我最不确定的地方：我不知道黑洞视频会不会太压抑。
我希望 Codex 重点看：哪个视频最适合做首页首屏。
```

## 候选 1：黑洞边缘慢速旋转

用途：

- 首页第一屏主背景。

优点：

- 震撼感强。
- 中间偏暗，适合放标题。
- 转黑白后仍然有层次。

风险：

- 如果对比太高，文字可能显得刺眼。
- 运动太快会让人疲劳，需要选慢镜头。

评分：

| 指标 | 分数 |
|---|---:|
| 震撼感 | 20/20 |
| 中心留白 | 17/20 |
| 黑白效果 | 18/20 |
| 循环友好 | 12/15 |
| 网页格式 | 15/15 |
| 版权清楚 | 10/10 |
| 总分 | 92/100 |

结论：最推荐。

## 候选 2：银河星空延时

用途：

- 首页背景或备用背景。

优点：

- 安静耐看。
- 不容易影响标题。
- 循环比较自然。

风险：

- 震撼感不如黑洞。
- 如果星点太密，容易显得普通。

评分：

| 指标 | 分数 |
|---|---:|
| 震撼感 | 14/20 |
| 中心留白 | 19/20 |
| 黑白效果 | 17/20 |
| 循环友好 | 15/15 |
| 网页格式 | 15/15 |
| 版权清楚 | 10/10 |
| 总分 | 90/100 |

结论：最稳，适合备用。

## 候选 3：地球旋转

用途：

- 第二屏或关于区背景。

优点：

- 科技感强。
- 真实感强。

风险：

- 太常见，个人主页可能不够独特。
- 地球主体容易抢标题。

评分：

| 指标 | 分数 |
|---|---:|
| 震撼感 | 16/20 |
| 中心留白 | 10/20 |
| 黑白效果 | 16/20 |
| 循环友好 | 12/15 |
| 网页格式 | 15/15 |
| 版权清楚 | 10/10 |
| 总分 | 79/100 |

结论：可用，但不适合第一屏主背景。

## 最终选择

主背景：候选 1。  
备用图：候选 2 的关键帧。  
第二屏素材：候选 3。
