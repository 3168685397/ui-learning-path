# 作业提交模板

复制下面这段，每次交作业时填写。

```text
课程：
日期：
成果链接/截图：

自评分：

我完成了：
1.
2.
3.

我最不确定的地方：

我希望你重点看：

我觉得最像“高级黑白风”的地方：

我觉得最不像的地方：
```

## 示例

```text
课程：Lesson 01
日期：2026-06-22
成果链接/截图：submissions/day-01/

自评分：72/100

我完成了：
1. 找了 3 张黑白网页参考
2. 标出了标题、按钮、留白、明暗区域
3. 写了每张图的观察

我最不确定的地方：
我不知道“留白”是不是标对了。

我希望你重点看：
帮我判断我说的高级感是不是太空泛。
```
