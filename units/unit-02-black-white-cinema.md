# Unit 02：黑白高级风格

目标：做出克制、干净、有电影感的黑白视觉。

## 黑白风格不是只有黑和白

推荐灰度层级：

| 用途 | 颜色 |
|---|---|
| 背景黑 | `#050505` |
| 深灰 | `#121212` |
| 分割线 | `#2a2a2a` |
| 次要文字 | `#9a9a9a` |
| 主要文字 | `#f2f2f2` |
| 强调白 | `#ffffff` |

## 关键规则

- 大标题可以很大，但页面内容要少。
- 不要同时使用很多字体。
- 背景视频要暗，不要抢标题。
- 白色不要铺满全屏，留给重点。
- 按钮可以很低调，边框、细线、微光即可。
