# Unit 01：视觉感知

目标：先学会“看”，再学会“做”。

## 你要掌握的 5 个词

1. 留白：页面不是越满越高级。
2. 对齐：元素之间要有隐形秩序。
3. 对比：大和小、亮和暗、粗和细。
4. 层级：先看什么，后看什么。
5. 统一：字体、颜色、间距不要乱跳。

## 每日练习

| 关卡 | 任务 | 成果 |
|---|---|---|
| 1-1 | 找 3 张喜欢的网页截图 | 放到 `assets/references/` |
| 1-2 | 标出最大标题和主要按钮 | 一张标注图 |
| 1-3 | 写出每张图高级在哪里 | 每张 3 句话 |
| 1-4 | 找出共同规律 | 5 条规律 |
| 1-5 | 复盘 | 写到 `progress.md` |

## 小测验

看到一个页面时，先问自己：

- 这个页面最先让我看到什么？
- 它哪里最亮？哪里最暗？
- 字体是不是太多？
- 按钮是不是很像按钮？
- 空白是不是让内容更清楚？
