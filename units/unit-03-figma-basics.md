# Unit 03：Figma 基础

目标：能用 Figma 画出首页设计稿。

## 只先学这些

- Frame：画桌面和手机画布
- Text：写标题和说明
- Rectangle：做按钮、线条、卡片
- Auto Layout：控制排列和间距
- Component：复用按钮和卡片
- Export：导出图片

## 练习路径

| 关卡 | 任务 | 成果 |
|---|---|---|
| 3-1 | 新建桌面 Frame：1440 x 900 | 空画布 |
| 3-2 | 放一个黑色背景 | 首屏底色 |
| 3-3 | 放大标题 | Hero 标题 |
| 3-4 | 做两个按钮 | 主按钮/次按钮 |
| 3-5 | 做 3 张作品卡片 | 卡片组件 |
| 3-6 | 做手机端 Frame | 390 x 844 |
| 3-7 | Boss：完成首页静态稿 | 一张完整设计图 |

## 自评

- 标题是否一眼能看到？
- 元素是否对齐？
- 间距是否统一？
- 手机端有没有拥挤？
- 黑白层级是否清楚？
