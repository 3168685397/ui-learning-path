# Unit 06：WordPress 发布

目标：把设计变成 WordPress 首页。

## 发布路线

1. 在 WordPress 媒体库上传背景视频和备用图。
2. 创建一个页面，命名为「首页」。
3. 使用 HTML 区块放入自定义代码。
4. 设置「首页」为静态主页。
5. 检查手机端。
6. 备份。

## 安全提醒

上线前先确认：

- WordPress 后台密码已改。
- 服务器 22 端口没有全网开放。
- 防火墙只开放网站需要的 80 和 443。
- 不把密码、授权码、API Key 写进仓库。
