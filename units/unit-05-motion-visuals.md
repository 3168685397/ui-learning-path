# Unit 05：动态视觉效果

目标：让页面有电影感，但不变得混乱。

## 优先学习的动效

| 动效 | 用途 | 难度 |
|---|---|---|
| 背景视频 | 制造沉浸感 | 低 |
| 黑白滤镜 | 统一风格 | 低 |
| 胶片颗粒 | 增加质感 | 低 |
| 标题淡入 | 让页面更有节奏 | 低 |
| hover 微动 | 提升可点击感 | 中 |
| 滚动视差 | 做出层次 | 中 |

## 不建议一开始做

- 太复杂的 3D
- 大量粒子
- 自动播放声音
- 页面里到处飞的动画

## 第一版动效标准

- 背景视频循环播放
- 视频加黑白滤镜
- 页面覆盖一层暗角
- 覆盖轻微噪点
- 标题进入时淡入上移
- 卡片 hover 有轻微位移

## Boss 关

做出一个可运行的首页首屏 Demo：

- 桌面端能看
- 手机端不崩
- 视频加载失败时有备用图
- 标题清晰可读
