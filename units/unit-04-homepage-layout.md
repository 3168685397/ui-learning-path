# Unit 04：网页首页布局

目标：理解个人主页由哪些区块组成。

## 推荐结构

1. Hero 首屏：背景视频 + 大标题 + 一句话说明。
2. About：一句短介绍。
3. Works：3 到 6 个作品卡片。
4. Notes：博客或动态。
5. Contact：邮箱、社交链接。

## 黑白宇宙主页首屏结构

```text
------------------------------------------------+
| LOGO                         WORKS ABOUT MAIL |
|                                                |
|            QUANSHEN                           |
|            Personal Space                     |
|                                                |
|            [View Works] [Contact]             |
|                                                |
|  01 / VISUAL    02 / AI    03 / LIFE          |
+------------------------------------------------+
```
