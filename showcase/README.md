# 案例展示

这里不是素材库，而是「看案例学判断」。

每个案例都包含：

- 适合学习什么
- 页面结构
- 高级感来源
- 可模仿的点
- 不能照搬的点
- 如果用于你的黑白宇宙个人主页，应该怎么改

## 看案例时不要做什么

- 不要只说“好看”。
- 不要急着找同款模板。
- 不要把案例的所有元素都搬过来。
- 不要为了炫动效破坏可读性。

## 看案例时必须写什么

每看一个案例，写 5 行：

```text
第一眼看到：
高级感来自：
我可以模仿：
我不能照搬：
放到我的主页要改：
```
