# 案例 02：黑白杂志式作品集

## 适合学习

- 如何用文字排版做高级感
- 如何展示作品而不显得杂乱
- 如何用黑白灰做层级

## 画面设定

页面像一本黑白杂志。左侧是大标题，右侧是个人介绍和作品编号。作品卡片不是彩色卡片，而是黑白截图、细线边框和小字号说明。

## 页面结构示意

```text
+--------------------------------------------------+
| INDEX                                   ABOUT     |
|                                                  |
| SELECTED                                         |
| WORKS                                            |
|                                                  |
|  01  AI Personal Homepage        2026            |
|  02  Visual Experiment           2026            |
|  03  WordPress Lab               2026            |
|                                                  |
|                         A quiet place for making |
+--------------------------------------------------+
```

## 高级感来源

| 来源 | 解释 |
|---|---|
| 杂志排版 | 文字也能成为视觉 |
| 编号系统 | 让作品展示更有秩序 |
| 细线分割 | 比厚重卡片更克制 |
| 大小对比 | 大标题和小说明形成层级 |
| 留白 | 页面不急着塞满 |

## 可模仿的点

- 作品不要都做成圆角卡片。
- 用编号和年份建立秩序。
- 用细线和灰字代替复杂装饰。
- 每个作品只写一句话。

## 不能照搬的点

- 如果全是文字，会缺少你想要的震撼视觉。
- 如果字体没选好，会像普通文档。
- 如果作品太少，页面可能显空。

## 放到你的主页怎么改

这个案例适合作为首页第二屏：

```text
SELECTED WORKS
01  黑白宇宙个人主页
02  AI 工具学习记录
03  WordPress 建站实验
```

首屏仍然用宇宙视频，第二屏用杂志式作品列表。

## 评分示范

| 指标 | 分数 | 原因 |
|---|---:|---|
| 黑白统一 | 20/20 | 完全黑白灰 |
| 标题可读 | 18/20 | 层级清楚 |
| 震撼感 | 12/20 | 更安静，不够冲击 |
| 高级感 | 19/20 | 留白和文字秩序很好 |
| 可实现性 | 20/20 | 很容易网页化 |

总分：89/100。
