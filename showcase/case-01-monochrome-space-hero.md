# 案例 01：黑白深空首页

## 适合学习

- 宇宙背景怎么放标题
- 黑白电影感怎么做
- 大标题和暗背景如何保持可读
- 少元素也能有冲击力

## 画面设定

一个深空背景视频占满整个首屏。背景是缓慢移动的星云或黑洞边缘，整体转成黑白。中间偏左放一个巨大标题，右下角放一个低调按钮，底部放 3 个小标签。

## 页面结构示意

```text
+--------------------------------------------------+
| QN                                      WORK MAIL |
|                                                  |
|                                                  |
|        QUANSHEN                                  |
|        PERSONAL SPACE                            |
|        Exploring AI, design and life.            |
|                                                  |
|        [View Works]   [Contact]                  |
|                                                  |
|  01 / AI       02 / DESIGN       03 / LIFE       |
+--------------------------------------------------+
```

## 高级感来源

| 来源 | 解释 |
|---|---|
| 大面积黑 | 让画面有深度，减少廉价感 |
| 大标题 | 快速建立视觉焦点 |
| 少按钮 | 避免像后台系统 |
| 低饱和 | 黑白灰统一气质 |
| 慢运动背景 | 有电影感，不打扰阅读 |
| 底部小标签 | 增加信息层次，但不抢主标题 |

## 可模仿的点

- 背景视频全屏铺满。
- 标题不居中死板，可以偏左。
- 按钮用白色细边框，不用花哨渐变。
- 底部信息用小字号、大字距。
- 加暗角和噪点，让画面更像电影。

## 不能照搬的点

- 不要让宇宙背景太亮。
- 不要把标题放在星云最亮处。
- 不要加太多技术标签。
- 不要自动播放声音。

## 放到你的主页怎么改

你的第一版可以这样写：

```text
QUANSHEN
PERSONAL SPACE
在 AI、设计和生活之间，搭建自己的数字宇宙。
```

底部标签：

```text
AI TOOLS
BLACK WHITE UI
DAILY NOTES
```

## 评分示范

| 指标 | 分数 | 原因 |
|---|---:|---|
| 黑白统一 | 18/20 | 色彩克制，灰度清楚 |
| 标题可读 | 19/20 | 标题足够大，背景压暗 |
| 震撼感 | 18/20 | 深空背景有尺度感 |
| 高级感 | 17/20 | 元素少，留白舒服 |
| 可实现性 | 18/20 | WordPress + HTML/CSS 可实现 |

总分：90/100。

## 练习问题

1. 如果背景视频中间太亮，你会怎么处理？
2. 如果标题看不清，是先换字体，还是先压暗背景？
3. 这个案例最多能放几个按钮？
