# 案例 03：电影感产品展示页

## 适合学习

- 如何把页面做得像短片
- 如何用滚动讲故事
- 如何让数据、标签、按钮叠在背景上

## 画面设定

页面第一屏是大幅黑白视频背景，第二屏出现数据和标签，第三屏出现说明文字和按钮。滚动时背景缓慢移动，文字逐层淡入。

## 页面结构示意

```text
第一屏：大标题 + 视频背景
第二屏：3 个数据 + 3 个标签
第三屏：简短说明 + 行动按钮
```

## 高级感来源

| 来源 | 解释 |
|---|---|
| 滚动叙事 | 页面像一个小短片 |
| 慢节奏 | 不急着把所有信息倒出来 |
| 数据点缀 | 增加真实感 |
| 动效克制 | 淡入、位移、视差即可 |
| 背景统一 | 所有页面像同一个世界 |

## 可模仿的点

- 首页不要一次展示全部内容。
- 每滚动一屏只出现一个主题。
- 数据可以用来增加视觉节奏。
- 动效要慢，不要弹来弹去。

## 不能照搬的点

- 如果没素材，硬做电影感会很空。
- 如果动效太多，低配置手机会卡。
- 如果文字太小，移动端看不清。

## 放到你的主页怎么改

你的网页可以分成：

```text
第一屏：QUANSHEN / PERSONAL SPACE
第二屏：AI / DESIGN / LIFE 三个方向
第三屏：作品列表
第四屏：联系方式
```

## 评分示范

| 指标 | 分数 | 原因 |
|---|---:|---|
| 黑白统一 | 18/20 | 背景和文字统一 |
| 标题可读 | 16/20 | 动态背景可能干扰文字 |
| 震撼感 | 20/20 | 电影感强 |
| 高级感 | 17/20 | 取决于动效是否克制 |
| 可实现性 | 14/20 | 需要更多代码和性能优化 |

总分：85/100。
